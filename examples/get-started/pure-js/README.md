## Example: Use hubble.gl without React and Base Map

Based on [deck.gl pure-js examples](https://github.com/visgl/deck.gl/tree/master/examples/get-started/pure-js).
Uses [Webpack](https://github.com/webpack/webpack) to bundle files and serves it
with [webpack-dev-server](https://webpack.js.org/guides/development/#webpack-dev-server).

## Usage

To install dependencies:

```bash
npm install
# or
yarn
```

Commands:
* `npm start` is the development target, to serve the app and hot reload.
* `npm run build` is the production target, to create the final bundle and write to disk.
